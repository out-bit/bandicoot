Installation
===========

```shell

```

or

```shell
easy_install outbit
```

Usage
===========

### Usage

$ outbit
  Username: xxxxxx
  Password: xxxxxx

user@outbit> help
  webserver
  apps
  useraccounts
 
user@outbit> help webserver
  webserver restart  [all|group|host]
