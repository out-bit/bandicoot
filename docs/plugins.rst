outbit Plugins
==================

The plugins command will show plugins that are available to use in user defined actions.

.. sourcecode:: bash
    outbit> plugins list
      roles_list
      secrets_edit
      help
      actions_edit
      logs
      actions_add
      users_list
      roles_add
      secrets_add
      roles_edit
      users_del
      actions_del
      roles_del
      secrets_list
      command
      users_edit
      ping
      users_add
      secrets_del
      plugins_list
